dyad {waved}R Documentation

Dyadic band

Description

Returns a vector of integers $2^j+1,...,2^{j+1}$.

Usage

dyad(j)

Arguments

j Resolution Level

Value

Returns a vector of integers $2^j+1,...,2^{j+1}$.

Marc Raimondo
